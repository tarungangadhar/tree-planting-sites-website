Abstract

This project identifies optimal urban tree planting sites in Buffalo, NY using the Bureau of Forestry Tree Inventory dataset (133,000+ records). We applied K-Means clustering, regression models, and decision tree classification combined with geospatial visualization (heatmaps & interactive maps) to recommend zones for new plantations. Results highlight priority areas with low eco-benefits where strategic tree planting can maximize environmental impact.
